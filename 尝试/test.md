```
她内心很惶恐，这一切的一切，或者说是在他人眼中的一切的一切都是假的。表面的荣光倒是要感谢幼时堆起的一张张笑脸。胃里一阵翻滚，所有的令人生冷汗的情绪就快要随着胃里的内容物翻涌出来，她还是挂着笑容。
谁能救救她呢。此时此刻，她无比希望自己有什么信仰，希望有所谓的救世主正俯视着历经磨难的信徒，希望这位救世主能大发慈悲为她指一条明路。
-- 是的，在某些人眼中，按部就班、一潭死水般的生活是她可望而不可即的，也是她唾弃的。她为自己争取来的生活感到欣慰，也为这种生活带来的不确定性辗转反侧。
```

```
当代快节奏的生活，让她陷入了漩涡。她在漩涡的中心无法自拔，只能随着人流继续下陷。她不记得已经有多久没有审视过自己了，她不记得好多事物的由头。“没事的，过去的就让它过去吧”，这种得过且过的念头让她羞愧，可她真的不记得了。一开始羞愧还能让她脑热脸红，“下次争取记得哦”，现在则是一个轻飘飘的念头。
阈值的提高让她不再有灵气，慢慢变得行尸走肉起来。
```

```
Be slient
是她对情人唯一的要求。
压力总是要宣泄的，以前宣泄对象是父母，现在则是情人。
她喜欢情人赤身裸体地在目所能及的地方，这让她感到存在与安全。
在南方的梅雨季待久了，她对干爽的环境是颇有偏爱的。即使在情动时也是如此。
```

```

```
故事是怎样发生的呢，大概就是在一个阴冷的雨天，去帮老板送合同。
"哎……"。
少年人少叹气，这是刚参加工作时前辈的教诲。
说真的，她不想出门。早上刚在公司结束述职，下午是想给自己放假的。已经不是刚工作时的年岁，没有那么多的干劲，贤者时间也被无限拉长。她只想买杯热饮，开车回家，开着空调，盖着被子，睡上一觉。
车上的收音机提醒她前方道路拥堵。
"哎……"。
这只是一个平常周六的午后，天气又不好，哪来那么多人。她拍了下方向盘，发泄着心中的烦闷。昨天晚上就该把合同交了的，说到底还是自己懒。车上的空调和停滞的车流让她有点微微发汗。
"看样子是要迟到了，希望顺利"。
到达目的地，一个城郊的会所，她拿起副驾上的文件和包，冲进了雨里，跑到了廊下。她今天穿得很正式，一身套装。
工作述职嘛，这种抢头花的事，当然是希望别人注视的。她人不高，穿上高跟才能过一米六的门。高腰的裙装，外加偏长的高跟鞋让她从视觉上长了那么几公分。
她在廊下拂去身上的水珠，推门进去。
今天合同的交接地点在甲方的地盘，一个酒吧。午后，绝对不是酒吧的目标营业时间。

（冬雨不像是春雨细如牛毛又给人生机，只有阴沉。）
